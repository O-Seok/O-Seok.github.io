# print()

파이썬의 print() 함수는 출력 함수이다.
간단히 문자열을 출력해본다.
그 유명한 Hello World 를… 😁

기본출력

문자열은 다음과 같이도 출력 가능하다.

Escape 코드
|코드|특징|
|–|–|
|\n| 개행|
|\t| 탭|
|\\ |문자|
|\‘ |문자|
|\“ |문자|
|\r |캐리지 리턴|
|\f |폼 피드|
|\a | 벨 소리|
|\b |백 스페이스|
|\000 |널 문자|

# 옵션

print()함수는 출력 값에 다양한 옵션을 줄 수 있다.

separator

값들 사이에 옵션을 넣어줄 수 있다.

end

값 끝에 옵션을 줄 수 있다.

format

format 옵션을 이용하여 다양하게 특정한 출력 값들을 매칭 시켜줄 수 있다.

%s : 문자, %d = 정수, %f = 실수
